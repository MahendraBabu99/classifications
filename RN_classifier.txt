Radius Neighbors Classifier
Overview

The Radius Neighbors Classifier is a distance-based, supervised classification algorithm.
It assigns a class label to a data point based on the labels of all training samples within a fixed radius around that point.

It is a variant of K-Nearest Neighbors (KNN), but instead of using a fixed number of neighbors, it uses a fixed distance threshold.

Core Idea

A data point belongs to the class that is most common among all training points lying within a specified radius.

When to Use Radius Neighbors Classifier

Data has varying density

You want local influence only

You want to ignore far-away points

Clusters have meaningful spatial boundaries

Advantages

Simple and intuitive

Adapts to local data density

No training phase (lazy learning)

Flexible distance metrics

Limitations

Sensitive to radius selection

Poor performance in high-dimensional data

No prediction if neighborhood is empty

Computationally expensive for large datasets

Common Applications

Spatial data classification

Anomaly-aware classification

Pattern recognition

Small to medium-sized datasets
