Linear Discriminant Analysis (LDA)
1. Definition

Linear Discriminant Analysis (LDA) is a supervised machine learning algorithm used for classification and dimensionality reduction. It projects data onto a lower-dimensional space while maximizing class separability.

2. Objective

LDA aims to find a linear transformation that:

Maximizes between-class variance

Minimizes within-class variance

This ensures that samples from different classes are as far apart as possible, while samples within the same class are tightly clustered.

3. Assumptions

LDA is based on the following assumptions:

Data for each class follows a Gaussian (normal) distribution

All classes share the same covariance matrix

Features are continuous

Observations are independent

6. Classification Rule

After projection:

Each class is modeled using a Gaussian distribution

A new sample is assigned to the class with the highest posterior probability

8. When to Use LDA

Use LDA when:

You have labeled data

Class distributions are approximately Gaussian

Classes are linearly separable

You want both dimensionality reduction and classification

9. Limitations

Performs poorly if covariance matrices differ significantly

Not suitable for highly non-linear class boundaries

Sensitive to outliers

10. Common Applications

Face recognition

Medical diagnosis

Text classification

Pattern recognition
